BZOJ 3170 松鼠聚会

题目给出的距离实际上是切比雪夫距离，即dist = max(|x1 - x2|, |y1 - y2|)。曼哈顿距离是可以转化为切比雪夫距离的，对于曼哈顿距离|x1 - x2| + |y1 - y2|，当我们变换坐标(x, y)为(x + y, x - y)后，其切比雪夫距离为max(|x1 + y1 - x2 - y2|, |x1 - y1 - x2 + y2|)，通过分类讨论可以证明两者是等价的。

那么在这道题里，我们变换坐标(x, y)为((x + y) / 2, (x - y) / 2)即将切比雪夫距离转化为了曼哈顿距离，从而实现了x和y坐标的分离统计，这个只要排序后扫描就可以了，时间复杂度O(NlogN)。
