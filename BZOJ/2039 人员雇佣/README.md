BZOJ 2039 人员雇佣

分集合的问题，很容易想到最小割。要使收益最大，那我们先把所有收益相加（即sigma{e_(i, j)}），再扣除构成合法方案需要的最小花费。

我们知道，花费只从用于分集合的与源汇相连的边和人与人之间的边来体现。

由于此题中任何两个人之间都有边，考虑以下三种情况：

A被雇佣，B也被雇佣，则需要的花费为雇佣A的代价 + 雇佣B的代价；

A被雇佣，B未被雇佣，则需要的花费为雇佣A的代价 + 4 * e_(A, B)；

A未被雇佣，B也未被雇佣，则需要的花费为2 * e_(A, B)。

那么，A被雇佣的花费为雇佣的代价a_A，A未被雇佣的花费为sigma{e_(A, i)}，被雇佣的人A与未被雇佣的人B之间的边的花费为2 * e_(A, B)，据此建图就很方便了。

时间复杂度O(N^4)。
