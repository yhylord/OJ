BZOJ 2132 圈地计划

分集合的问题，很容易想到最小割。

商业区和工业区对应A类点和B类点。

题目的瓶颈在于，如果直接把两类点作为两个集合，相邻点连边，那么割代表的是异类相邻产生的收益，最小割即代表最小收益，恰恰与题意相反。

由于这是二分图，需要利用其特殊性质――X集合或Y集合内部无边。

我们预先把所有的收益都累加起来，一种合法方案即对应扣除其中一些收益，要收益最大也即扣除的收益最小。

那么，我们令S集合为二分图中的X集合中的A类点和Y集合中的B类点，T集合为二分图中的X集合中的B类点和Y集合中的A类点，那么割掉的边恰恰是需要扣除的同类边。

由此，题目被转化为了一个最小割问题，时间复杂度O((NM)^3)。
