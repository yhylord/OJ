BZOJ 3156 防御准备

用f(i)表示到第i个点为止（第i个点必建立检查点）所需最小总代价，则f(i) = min{f(j) + (i - j) * (i - j - 1) / 2} + a_i(0 <= j < i)。

变形得f(i) = min{f(j) + j * (j + 1) / 2 - i * j} + a_i + i * (i - 1) / 2。

令x = j, y = f(j) + j * (j + 1) / 2，我们要min z = -ix + y，也即要使直线y = ix + z的纵截距最小。

由于x单调增，斜率i单调增，我们可以用单调队列维护下凸壳来完成斜率优化。时间复杂度O(N)。
