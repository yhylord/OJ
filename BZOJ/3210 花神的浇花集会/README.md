BZOJ 3210 花神的浇花集会

我们首先变换坐标(x, y)为(x + y, x - y)即将切比雪夫距离的2倍转化为了曼哈顿距离，从而实现了x和y坐标的分离统计。

以x坐标为例，易证明最优位置必然是已存在的某个x坐标，因此只要排序后扫描就可以了。

然而要注意的是，我们将坐标放大了2倍，新坐标(x, y)还原后的坐标实际为((x + y) / 2, (x - y) / 2)，这样我们x和y分别得到的最优答案形成的点原来可能不是整点。

因此我们必须对求出的最优坐标进行抖动，易证明只要加减1即可。

那么，我们只要处理出x坐标在奇数和偶数下的最优解以及y坐标在奇数和偶数下的最优解，同奇偶相加取最优即可。

时间复杂度O(NlogN)。
