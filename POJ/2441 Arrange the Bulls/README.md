POJ 2441 Arrange the Bulls

【题目大意】

有N(1 <= N <= 20)头牛和M(1 <= M <= 20)个牧场，每头牛都有若干个喜欢的牧场，它们都不喜欢和别的牛在同一个牧场，问有多少种分配方案使得每头牛都有自己的牧场。

【算法分析】

这是个和二分图匹配有关的模型，要求一个集合的点都被匹配。

一个显然的想法是用f[i][j]表示到第i头牛为止使用的牧场集合为j的方案数，然后通过枚举第i头牛放在哪个集合来转移，但这样显然太过浪费了。

我们发现，集合j的大小（即1的个数）恰好可以表示当前已经到第几头牛了，所以我们只要记状态f[i]表示使用牧场集合为i的方案数，i从小到大枚举，判断是第几头牛再同上转移。

时间复杂度O(M * 2^M)。
