POJ 1113 Wall

【题目大意】

给出一个N(3 <= N <= 1000)个点的多边形和L(1 <= L <= 1000)，要求造一圈围墙，且墙到多边形上任意点的距离都不小于L，求墙的最小长度。

【算法分析】

墙必然是一个圆滑的凸形，因为凹进来只会使答案更劣。由此，我们只需要求出这N个点的凸包，再在凸包外L的距离围一圈即可。

我们发现，墙的长度恰好等于凸包的周长加半径为L的圆的周长，因为除了凸包的平行线就是拐角处的圆弧，由多边形外角和360度可知合起来构成一个圆。

那么，我们用Graham扫描法求凸包求可以了，注意极角排序时处理共线点的顺序，以及圆周率的精度。时间复杂度O(NlogN)。
