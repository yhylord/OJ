POJ 3680 Intervals

【题目大意】

给出N个开区间，第i个区间(a_i, b_i)价值w_i，要求选若干区间使得没有一个点被覆盖超过K(1 <= K <= N <= 200)次，且价值最大。

【算法分析】

在脑海中想象出一根数轴，把所有区间平行摆放于数轴上方，同时添上i到i + 1的虚边，我们会发现题目就是要求不超过K条不相交路径使得费用最大。

因此，我们先将坐标离散化，设最大坐标为M，将源点向1连容量K费用0的边，将M向汇点连容量K费用0的边，将i到i + 1连容量正无穷费用0的边，将区间端点a_i到b_i连容量1费用w_i的边，求最大费用最大流即可。时间复杂度O(NMU)。
