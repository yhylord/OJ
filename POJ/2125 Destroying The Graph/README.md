POJ 2125 Destroying The Graph

【题目大意】

给出一个N个点(1 <= N <= 100)M条边(1 <= M <= 5000)的有向图，每次可以用a_i的代价删除点i的所有入边或用b_i的代价删除点i的所有出边，求用最少的代价删除所有的边。

【算法分析】

把每个点拆为入点和出点，将出点作为X集合，入点作为Y集合，原图中的边在新图中从出点指向入点，则我们建出了一个二分图，而我们要求的就是用最少的点权和覆盖所有的边，即二分图最小点权覆盖问题。

我们新建源点和汇点，将源点向X集合连容量为点权的边，将Y集合向汇点连容量为点权的边，将原图中的边容量设为无穷大，那么就可以利用最小割解决。时间复杂度O(MN^2)。
