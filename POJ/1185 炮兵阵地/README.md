POJ 1185 炮兵阵地

【题目大意】

给出一个N * M(1 <= N <= 100, 1 <= M <= 10)的棋盘，一些格子不能放置棋子。求最多能在棋盘上放置多少个棋子，使得每一行每一列的任两个棋子间至少有两个空格。

【算法分析】

首先我们用DFS求出一行的可能状态（即彼此至少隔2格）共S种（经计算M = 10时S = 60），并记录下对应的1的个数。

那么，我们用f[i][j][k]表示到第i行为止，第i行状态为第j种，第i - 1行状态为第k种最多能放多少个棋子。

我们只要枚举第i - 2行的状态，且这三行的状态都满足对应行的禁止要求，然后两两之间and的结果都为0即可转移。

时间复杂度O(NS^3)。
