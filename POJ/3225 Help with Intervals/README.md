POJ 3225 Help with Intervals

【题目大意】

定义一些区间操作（详见原题描述），要求求出执行一系列区间操作以后得到的区间。

【算法分析】

首先，由于题目没有给出数的范围，我们必须先对这些数进行离散化。
得到新数后，要处理开闭区间问题，这也不难，我们把每个数拆成三个数，分别表示小于这个数、这个数、大于这个数的三段。
那么，对于每个操作，我们可以找到转化后的左右端点l, r，接下来就是考虑进行操作的问题了。
U：l到r覆盖为1；
I：l之前和r之后覆盖为0；
D：l到r覆盖为0；
C：l之前和r之后覆盖为0，l到r取反；
S：l到r取反。
因此，可以用线段树来进行维护，时间复杂度O((N + Q)logN)。
