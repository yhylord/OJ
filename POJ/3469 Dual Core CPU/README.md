POJ 3469 Dual Core CPU

【题目大意】

给出N个点(1 <= N <= 20000)，M条边(1 <= M <= 200000)，将这些点分成两个集合A, B，点i若属于集合A则需要代价a_i，若属于集合B则需要代价b_i，不在同一集合里且有边的两个点i, j需要代价w(i, j)，求最小代价。

【算法分析】

分割成两个集合且代价最小是经典的最小割问题。暂时不考虑集合间边的代价，仅仅分配每个点所在的集合，我们将源点向点i连容量为a_i的边，将点i向汇点连容量为b_i的边，那么每个点的这两边中必然有且仅有一条边在最小割中，即对应一种分配方案，T集合即A集合，S集合即B集合。

同时，我们可以发现，在同一个集合里的点之间是不需要割开的，故不可能存在割边的。那么我们将原图中的无向关系边正向反向均加入新图中，不在同一集合里的两点间必须要割掉此边。由此，新图的一个割就对应了一种合法方案的代价，最小割即为最小代价。时间复杂度O(MN^2)。
