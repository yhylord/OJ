POJ 1038 Bugs Integrated, Inc.

【题目大意】

给出一个N * M(1 <= N <= 150, 1 <= M <= 10)的棋盘，要在上面放置2 * 3的骨牌，且给出K(0 <= K <= N * M)个无法放置的格子，求最多能放置多少个。

【算法分析】

我们用f[i][j]表示前i - 2行覆盖满，第i - 1行和第i行状态为j（0表示均留空，1表示第i - 1行填满第i行留空，2表示均填满）最多能放几个。

那么，我们可以用DFS求出所有合法的转移关系，对于某一格有两种摆放和三种留空方式，注意细节即可。

无法摆放的位置可以事先处理一下，时间复杂度O(NM + K + N * 3^M)。
