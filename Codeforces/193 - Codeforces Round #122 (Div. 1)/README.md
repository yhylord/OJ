193 - Codeforces Round #122 (Div. 1)

C. Hamming Distance

【题目大意】

求最短的4个由a, b组成的等长字符串s_1, s_2, s_3, s_4，使得s_i和s_j对应位不同的个数满足给出的h(i, j)(0 <= h(i, j) <= 10^5)。

【算法分析】

由于题目给出了6个限制条件，我们想到要去列6个方程来解。

考虑每一位的不同状态其实只有7种："aaab", "aaba", "abaa", "baaa", "aabb", "abba", "abab"（"aaaa"只会增加无谓的长度）。

于是，用x_i表示状态i的个数，7个变量，6个方程，我们可以先通过高斯消元法进行处理。

由于h(i, j) <= 10^5，那么每个x_i <= 10^5（若超过则h(i, j)必有一超过），可以对自由元x_7进行枚举再进行回代，求出总长的最小值。整体时间复杂度O(max{h(i, j)})。
