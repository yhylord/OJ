297 - Codeforces Round #180 (Div. 1)

A. Parity Game

【题目大意】

给出两个01串A, B(1 <= |A|, |B| <= 1000)，对于A串，每次可以选择删除首字母，或是在末尾添加0（串中有偶数个1时）或1（串中有奇数个1时）。

【算法分析】

我们首先分析串中的1，可以发现，除了一开始A串中有奇数个1可以添加1个1外，此后串中1的个数只能减少不能增加。

现在我们已经有偶数个1了，那么对于B串我们依次构造，若为0就直接在A串末尾添加，若为1则删去A串的一个前导1添到末尾，又恢复了偶数个1的状态。

最后删去那些多余的前导1即可。由此，只要B串中1的个数不超过A串中1的最大个数就一定有解。时间复杂度O(|A| + |B|)。

B. Fish Weight

【题目大意】

有K(1 <= K <= 10^9)种鱼，第i种鱼的重量w_i满足0 < w_1 <= w_2 <= ... <= w_k。现在A买了N(1 <= N <= 10^5)条鱼，B买了M(1 <= M <= 10^5)条鱼，告诉你买的鱼的种类，判断是否存在一种鱼的重量的分配方案，使得A的鱼的总重量严格大于B。

【算法分析】

由于A, B鱼条数不等，我们首先可以给少的那个人补充重量为0的鱼使得两者条数相等。考虑将两个人的鱼的种类分别排序，若A的每一条鱼的种类都小于等于B，那么必然是不可能存在可行方案的。

假如有一个大于B，那么我们可以在这里弥补之前的所有损失，并使得此后所有的鱼重量都相等，从而实现严格大于。时间复杂度O(NlogN + MlogM)。
