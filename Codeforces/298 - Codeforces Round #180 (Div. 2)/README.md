298 - Codeforces Round #180 (Div. 2)

A. Snow Footprints

【题目大意】

有一条N(3 <= N <= 1000)个格子排成一排形成的路，初始时有一个石块，每次可以将石块左移或右移，同时会在上一个格子上留下左/右的痕迹，新痕迹可以覆盖旧痕迹。给出痕迹，求一组可行的起点和终点。

【算法分析】

我们观察发现，痕迹只可能有三种：全L，全R，先一段R后一段L。这几种情况分别构造即可。时间复杂度O(N)。

B. Sail

【题目大意】

给出起点和终点的坐标及接下来T个时刻的风向（东南西北），每次可以选择顺风偏移1个单位或者停在原地，求到达终点的最少时间。

【算法分析】

我们发现风向是互不影响的，而我们对四个方向的偏移量的要求是固定的，因此只要恰好取到满足四个偏移量的要求就可以了。时间复杂度O(T)。
