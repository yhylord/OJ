ZOJ 2676 Network Wars

【题目大意】

给出一个N个点(2 <= N <= 100)M条边(1 <= M <= 400)的带权无向图，求将点1和点N分开的一个边割集，使得该边割集的平均边权最小。

【算法分析】

首先，这是一个0-1分数规划问题，常用的手段是二分答案k，求构造函数的最小值，详见胡伯涛的《最小割模型在信息学竞赛中的应用》。之后，我们将所有边权减去k，负权边必取，其余边取最小割，判定答案。

注意到题中要输出方案，采用在残余网络中BFS出点1所在集合S，取集合S中的点引出到集合S外的边即可（必取的边必然在其中）。

还有一个是二分精度的控制问题，同样在胡伯涛论文后面有分析，设边集E1答案大于E2答案，设边权和分别为S1, S2，边数分别为N1, N2，则S1 / N1 - S2 / N2 = (S1N2 - S2N1) / N1N2 >= 1 / N1N2 >= 1 / N^2。

因此，由于此题M最多为400，精度控制到10^-6即可。时间复杂度P(MN^2 * logK)。
